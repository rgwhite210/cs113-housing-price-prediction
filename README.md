# cs113-housing-price-prediction
## Housing Price Prediction with Linear Regression
A Python program that predicts housing prices based on square footage using linear regression, built with housing data inspired by **Contra Costa County, CA**.

## Requirements
```bash
pip install pandas matplotlib scikit-learn numpy
```

## How to Run
```bash
python housing_predictor.py
```

## How It Works
1. Loads housing data from `housing_data.csv`
2. Displays basic statistics and a scatter plot
3. Trains a linear regression model
4. Displays the regression line over the data
5. Prompts user to enter a house size and returns a predicted price
